معادلات تبدیل موجک سریع

معادلات تعریف‌کننده FWT

معادلات زیر معادلات تعریف‌کننده برای محاسبات تبدیل موجک سریع هستند.

معادلات اصلی

ضرایب موجک

$$W_{\psi}(j, k) = \sum_m h_{\psi}(m - 2k) W_{\phi}(j + 1, m)$$

ضرایب مقیاس‌بندی

$$W_{\phi}(j, k) = \sum_m h_{\phi}(m - 2k) W_{\phi}(j + 1, m)$$

نمایش با پیچش

$$W_{\psi}(j, k) = h_{\psi}(-n) * W_{\phi}(j + 1, n) \quad n = 2k, k \geq 0$$

$$W_{\phi}(j, k) = h_{\phi}(-n) * W_{\phi}(j + 1, n) \quad n = 2k, k \geq 0$$

ساختار چند مرحله‌ای

بانک فیلتر می‌تواند تکرار شود تا ساختارهای چند مرحله‌ای را برای محاسبه ضرایب DWT در دو یا چند مقیاس متوالی ایجاد نماید.

بانک تحلیل FWT

بانک تحلیل FWT شامل:

  • فیلترهای تحلیل برای تجزیه
  • کاهش نرخ نمونه‌برداری
  • امکان تکرار برای مقیاس‌های بیشتر

بانک ترکیبی FWT

بانک فیلتر ترکیبی:

  • ضرایب افزایش نرخ نمونه‌برداری توسط عمل پیچش با $h_{\phi}(n)$ و $h_{\psi}(n)$ فیلتر می‌شوند
  • با هم جمع می‌گردند تا تخمینی با مقیاس بالاتر تولید گردد
  • تخمین بهتری از دنباله $f(n)$ با دقت و جزئیات بیشتر ایجاد خواهد شد

بازسازی کامل

این فرایند ترکیب ضرایب:

  • می‌تواند به هر تعدادی از مقیاس‌ها بسط داده شود
  • بازسازی کامل دنباله $f(n)$ را تضمین می‌کند

شرایط وجود FWT

گرچه توابع پایه فوریه (یعنی منحنی‌های سینوسی) وجود FFT را تضمین می‌کند، وجود FWT به:

  • وجود توابع مقیاس‌بندی برای موجک‌های مورد استفاده
  • قابلیت تعامد تابع مقیاس‌بندی و موجک‌های متناظر
