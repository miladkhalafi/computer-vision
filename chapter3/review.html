موضوعات اصلی فصل

✓ 1. مقدمه و مفاهیم پایه
  • حوزه مکانی (Spatial Domain)
  • حوزه تبدیل (Frequency Domain)
  • تعریف پردازش حوزه مکانی: $g(x,y) = T[f(x,y)]$
  • همسایگی و روش اجرای پردازش
  • تعریف ارتقای تصویر
✓ 2. تبدیلات پایه شدت
  • نگاتیو تصاویر: $s = L - 1 - r$
  • تبدیلات لگاریتمی: $s = c \cdot \log(1 + r)$
  • تبدیل معکوس لگاریتمی: $s = c \cdot (e^r - 1)$
  • قانون توان: $s = c \cdot r^\gamma$
  • مثال کد MATLAB
✓ 3. تبدیلات تکه‌ای خطی
  • گسترش کنتراست (Contrast Stretching)
  • بخش‌بندی سطح شدت (Intensity Level Slicing)
  • آستانه‌گیری (Thresholding)
  • بخش‌بندی تصویر به صفحات بیتی (Bit-Plane Slicing)
✓ 4. پردازش هیستوگرام
  • تعریف هیستوگرام: $h(r_k) = n_k$
  • هیستوگرام نرمال‌سازی شده: $p(r_k) = \frac{n_k}{MN}$
  • تساوی‌سازی هیستوگرام: $s_k = (L-1) \sum_{j=0}^{k} p(r_j)$
  • مطابقت هیستوگرام (Histogram Matching)
  • کاربردهای هیستوگرام
✓ 5. عملگرهای گرادیان
  • تعریف ریاضی گرادیان: $\nabla f = [G_x, G_y]^T$
  • اندازه گرادیان: $|\nabla f| = \sqrt{G_x^2 + G_y^2}$
  • عملگر روبرتز (Roberts) - هسته $2 \times 2$
  • عملگر سوبل (Sobel) - هسته $3 \times 3$
  • کاربردهای گرادیان در بازرسی صنعتی
✓ 6. عملگر لاپلاسین
  • تعریف لاپلاسین: $\nabla^2 f = \frac{\partial^2 f}{\partial x^2} + \frac{\partial^2 f}{\partial y^2}$
  • هسته 4 نقطه‌ای و 8 نقطه‌ای
  • استفاده از لاپلاسین برای تیز کردن تصویر
  • مقایسه با گرادیان
  • مثال عملی: اسکن هسته‌ای استخوان
✓ 7. تکنیک‌های ارتقای تصویر
  • روش‌های ارتقای تصویر
  • روش ترکیبی (گرادیان + لاپلاسین)
  • مثال عملی ارتقای اسکن هسته‌ای استخوان
  • نکات مهم در ارتقای تصویر
  • خلاصه فصل و کاربردهای عملی
