عملگرهای مورفولوژی (Morphological Operators)

تعریف

مورفولوژی شاخه‌ای از پردازش تصویر است که بر اساس تئوری مجموعه‌ها و جبر بولی عمل می‌کند. این عملگرها برای تحلیل شکل و ساختار اشیاء در تصویر استفاده می‌شوند.

مفاهیم پایه

ساختار عنصری (Structuring Element)

یک مجموعه کوچک که برای بررسی همسایگی استفاده می‌شود:

  • معمولاً ماتریس کوچک (3×3, 5×5)
  • دارای یک نقطه مرکزی (origin)
  • می‌تواند اشکال مختلف داشته باشد (مربع، دایره، خط)

عملگرهای پایه

1. دایلیشن (Dilation)

$$A \oplus B = \{z | (\hat{B})_z \cap A \neq \emptyset\}$$

ویژگی‌ها:

  • اشیاء را بزرگ می‌کند
  • حفره‌های کوچک را پر می‌کند
  • لبه‌ها را گسترش می‌دهد

کاربرد: اتصال اشیاء جدا شده، پر کردن حفره‌ها

2. اروژن (Erosion)

$$A \ominus B = \{z | B_z \subseteq A\}$$

ویژگی‌ها:

  • اشیاء را کوچک می‌کند
  • برآمدگی‌های کوچک را حذف می‌کند
  • اشیاء کوچک را حذف می‌کند

کاربرد: حذف نویز، جداسازی اشیاء

3. باز کردن (Opening)

$$A \circ B = (A \ominus B) \oplus B$$

ابتدا اروژن، سپس دایلیشن

ویژگی‌ها:

  • حذف برآمدگی‌های کوچک
  • حفظ شکل کلی
  • جدا کردن اشیاء متصل

4. بستن (Closing)

$$A \bullet B = (A \oplus B) \ominus B$$

ابتدا دایلیشن، سپس اروژن

ویژگی‌ها:

  • پر کردن حفره‌های کوچک
  • اتصال اشیاء نزدیک
  • صاف کردن مرزها

عملگرهای ترکیبی

1. گرادیان مورفولوژیک

$$\text{Gradient} = (A \oplus B) - (A \ominus B)$$

برای تشخیص لبه‌ها

2. Top Hat

$$\text{Top Hat} = A - (A \circ B)$$

برای استخراج اشیاء روشن از پس‌زمینه تاریک

3. Black Hat

$$\text{Black Hat} = (A \bullet B) - A$$

برای استخراج اشیاء تاریک از پس‌زمینه روشن

خواص مهم

خاصیت Duality

$$(A^c \oplus B) = (A \ominus \hat{B})^c$$

خاصیت Idempotency

$$(A \circ B) \circ B = A \circ B$$
$$(A \bullet B) \bullet B = A \bullet B$$

کاربردها

  • حذف نویز: استفاده از opening و closing
  • تشخیص لبه: گرادیان مورفولوژیک
  • سگمنتیشن: جداسازی اشیاء
  • استخراج ویژگی: Top Hat و Black Hat
  • پردازش شکل: تحلیل ساختار اشیاء
  • فیلترگذاری: حذف یا حفظ اشیاء بر اساس اندازه

مثال عملی

حذف نویز از تصویر باینری:

  1. Opening برای حذف نویزهای کوچک
  2. Closing برای پر کردن حفره‌ها

جداسازی اشیاء متصل:

  1. Erosion برای جدا کردن
  2. Dilation برای بازگرداندن اندازه
